Export-Oriented Agro Value Optimization System

IEEE Conference Publication – SASIGD 2026

A machine learning-based decision support system that helps farmers maximize profit by recommending crops, predicting yield, forecasting market prices, and suggesting whether to sell in local or export markets. The system integrates agricultural and economic analysis into a single workflow.

Features
1. Crop Recommendation
2. Yield Prediction
3. Price Prediction
4. Export vs Local Market Recommendation
5. Profit Optimization
6. High-value Crop Suggestions
